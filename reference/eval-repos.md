# Suggested Eval Repos

Use at least 3 repositories with different languages and sizes.

- `https://github.com/octocat/Hello-World` (tiny / baseline)
- `https://github.com/expressjs/express` (medium / JavaScript backend)
- `https://github.com/pallets/flask` (medium / Python backend)
- `https://github.com/facebook/react` (large / JavaScript + tooling)
- `https://github.com/kubernetes/kubernetes` (very large / Go)
